# CloudWatch Dashboard Definitions

> **Note**: These are **design documents** — the dashboards described below have not yet been provisioned in CloudWatch. Use the deploy commands in each section to create them when ready.

Two CloudWatch dashboards for Plato observability:
1. **Business Dashboard** — Plato/Business namespace
2. **Quality Dashboard** — Plato/AIDLC + Plato/Agent namespaces

---

## 1. Business Dashboard

**Namespace**: `Plato/Business`

Widgets:
- Skill usage distribution (bar chart)
- DAU trend (line, 7 days)
- Session depth histogram
- Artifact generation counts
- Flow completion rate (number widget)

### Deploy Command

```bash
aws cloudwatch put-dashboard \
  --dashboard-name Plato-Business \
  --dashboard-body file://plato-business-dashboard.json
```

## 2. Quality Dashboard

**Namespaces**: `Plato/AIDLC`, `Plato/Agent`

Widgets:
- AIDLC funnel (bar chart: stages completed)
- Stage duration heatmap
- Rework rate by stage
- Spec completeness scores
- Invocation latency p50/p99

### Deploy Command

```bash
aws cloudwatch put-dashboard \
  --dashboard-name Plato-Quality \
  --dashboard-body file://plato-quality-dashboard.json
```
